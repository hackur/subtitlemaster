Subtitle Master foi criado por Wilker Lúcio e é um projeto de código aberto

O Subtitle Master depende de outros grandes projetos para poder funcionar, sendo eles:

  • SudDB: banco de dados de legendas com API de fácil acesso, além do prover os dados o dono do mesmo participou ativamente do Subtitle Master ao criar API's especiais das quais o Subtitle Master tira vantagem para melhorar suas buscas.
  • Open Subtitles: enorme banco de dados de legendas.
